# 微软大草台班子Bug不断

在 Win10 和 Win11交接之际，微软做出了一系列逆天操作，10月份后，Windows Bug 不断。

•  Windows 10 错误结束支持警告（2025年11月） 10月更新后，仍在支持期内的Windows 10系统显示“版本已结束支持”虚假警报，误导用户需立即升级。影响：所有Windows 10用户，尤其是企业环境。

•  Windows内核权限提升零日漏洞（CVE-2025-62215，2025年11月） 竞争条件漏洞允许本地攻击者提升至管理员权限，已被积极利用。影响：Windows系统内核，易被恶意软件利用。

•  Windows 11 localhost连接中断（KB5066835，2025年10月） 10月安全更新破坏localhost连接，导致本地托管应用无法访问网络，伴随安装失败和外设兼容问题。影响：Windows 11 24H2/25H2用户，尤其是开发者。

•  Windows恢复环境（WinRE）输入设备失效（KB5066835，2025年10月） 更新后，键盘和鼠标在WinRE中无法使用，阻碍导航恢复选项。影响：Windows 11所有版本，用户在系统崩溃时无法修复。

•  Windows 10 ESU注册失败（2025年10月） 结束支持后，部分系统无法注册Extended Security Updates (ESU)程序，导致无法获取安全补丁。影响：Windows 10企业用户。

•  Microsoft Copilot应用意外卸载（2025年3月，影响持续至11月） 安全更新导致Copilot app在Windows 10/11上自动卸载，无法正常使用。影响：依赖Copilot的AI功能用户。

•  WSUS远程代码执行漏洞（CVSS 9.8），允许攻击者通过更新基础设施注入恶意代码，总计172个漏洞。影响：Windows Server和客户端，易被APT利用。

•  Windows Server 2025蓝屏和安装挂起（2025年11月） 高逻辑处理器（>256）系统出现蓝屏、安装失败或启动延迟长达3小时。影响：Windows Server 2025企业服务器。

•  任务管理器无法关闭（2025年10月可选更新） 安装后，任务管理器窗口无法正常退出。影响：Windows 11用户日常操作。

•  BitLocker恢复屏幕触发（2025年10月更新） 安全更新导致部分系统进入BitLocker恢复模式，需恢复密钥。影响：加密驱动器用户。
