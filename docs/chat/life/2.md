# 5201314问题
总是有人说今天是1314年5月20日经过5201314天，其实用脑子想想都知道是不对的。

就算按一年400天，也要经过13000年，谁能活这么久（包包也不行）

不过，具体是哪一天呢？

使用了Python程序进行测试：
'''
def is_leap_year(year):
    if year % 4 != 0:
        return False
    if year % 100 != 0:
        return True
    return year % 400 == 0

months = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

def days_in_month(year, month):
    if month == 2 and is_leap_year(year):
        return 29
    return months[month]

year = 1314
month = 5
day = 20
total_days = 5201314

# Days to Jan 1 of next year
days_to_next_year = 365 - (31 + 28 + 31 + 30 + 20) + 1  # day of year is 31+28+31+30+20=140, 365-140+1? Wait
# Actually, from day 140, days to end of year: 365 - 140 = 225, but to Jan 1 next is 226 days added.

# Calculate day of year
doy = 31 + 28 + 31 + 30 + 20  # Jan to Apr +20 =31+28=59,59+31=90,90+30=120,120+20=140
days_in_year = 365  # not leap
days_to_end = days_in_year - doy  # 365-140=225, add 225 -> Dec 31
# add 226 -> Jan 1, 1315

days_to_next_jan1 = days_to_end + 1  # 226

if total_days >= days_to_next_jan1:
    total_days -= days_to_next_jan1
    year += 1
    month = 1
    day = 1
else:
    # add to current year
    # but since large, skip
    pass  # will handle in else

# Now add full years
while True:
    year_days = 365 + (1 if is_leap_year(year) else 0)
    if total_days >= year_days:
        total_days -= year_days
        year += 1
    else:
        break

# Now add remaining days to Jan 1, year
# day=1, month=1

while total_days > 0:
    dim = days_in_month(year, month)
    days_left_in_month = dim - day + 1
    if total_days >= days_left_in_month:
        total_days -= days_left_in_month
        day = 1
        month += 1
        if month > 12:
            month = 1
            year += 1  # though shouldn't happen
    else:
        day += total_days
        total_days = 0

print(f"{year}-{month:02d}-{day:02d}")
'''

结果是15555年2月5日，2025年代的AI验算也大体是这个结果（除了Gemini 2.5Pro计算为了26日）

由于Python默认的datetime库最高只支持到9999年，所以不能直接调用库来完成该任务。

这个结果比起2025年初好多了，当时除部分国外AI能够大概算出来是万年级别（甚至还不能精确到一万五千年左右）。国内的大部分要么直接趴窝需要么就是几千年（文心一言还说是2072年）

由于1314年格里高利历法还没有启用，这里假设格里高利反推回去向前生效。同时考虑闰年。
